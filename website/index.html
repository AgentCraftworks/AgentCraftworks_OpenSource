An open-source GitHub App that automates multi-agent software development workflows using webhook events, CODEOWNERS-based routing, and the Model Context Protocol (MCP).

View on GitHub Quick Start

4-State Handoff FSM

Deterministic pending → active → completed state machine with failed transitions from any non-terminal state.

CODEOWNERS Routing

Automatically routes work to the right agent based on file paths defined in your CODEOWNERS file.

Engagement Level Governance

Five-tier permission system — from Observer (read-only) to Full Agent Team (deploy & orchestrate) — with per-environment caps.

MCP Server

Six built-in Model Context Protocol tools for creating, accepting, completing, and querying agent handoffs.

Action Classification

T1–T5 action tier system that maps every GitHub action to the minimum engagement level required to perform it.

Secure Webhook Processing

Every incoming webhook is verified with HMAC-SHA256 before any business logic runs.

Agent Engagement Levels

Level Name Action Tier Permitted Actions
1 Observer T1 Read, view, list
2 Advisor T2 Comment, suggest
3 Peer Programmer T3 Label, assign, approve, edit file
4 Agent Team T4 Merge, close, create branch, push commit
5 Full Agent Team T5 Deploy, modify CI, orchestrate agents
